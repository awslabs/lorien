Contribute to LorienÂ¶

Lorien is a large project and cannot be done by a single person. As a result, we welcome everyone to contribute. We value all forms of contributions, including but not limited to:

  • Code reviewing of the existing patches.

  • Documentation and usage examples.

  • Code readability and developer guide.

    • We welcome contributions that add code comments to improve readability

    • We also welcome contributions to docs to explain the design choices of the internal.

  • Test cases to make the codebase more robust.

  • Tutorials, blog posts, talks that promote the project.
